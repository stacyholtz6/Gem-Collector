Gem Collector

Game Rules

  1. You will be given a random number at the start of the game, this is the Target Score.
  2. Each gem is give a random number. When a gem is clicked the number that gem was given will be added to the Current Score. It's up to you to figure out what numbers the gems were given.
  3. You win if the Current Score matches the Target Score.
  4. You lose if the Current Score is greater than the Target Score.
Good luck and have fun!!!
